# Nulls Messenger (UI MVP)

Локальный прототип мессенджера без сервера: список чатов, отправка сообщений, имитация ответа, модальные окна звонков и гибкая кастомизация профиля (имя, статус, аватар, цвет темы, фон чата).

## Запуск

```bash
python3 -m http.server 4173
```

Откройте в браузере: <http://localhost:4173>
